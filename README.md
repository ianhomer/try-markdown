# Try Markdown

Repository of markdown content for experimentation.

## Linting

    npm install

### remark

    npm run lint:remark
    npm run lint:remark:bad

### markdownlint

    npm run lintLmarkdownlint
    npm run lintLmarkdownlint:bad
